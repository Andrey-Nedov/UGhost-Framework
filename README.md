## Цель проекта

Создать кроссплатформенное ПО, которое позволит сторонним разработчикам, не имеющим особых знаний в области распознавания и синтеза речи, встраивать в свои проекты ботов, 
распознающих и синтезирующих речь. 

<img src="/imgs/anim.gif" width="600"/>

В ходе работы были проанализированы все доступные открытые и платные программные средства для реализации распознавания и синтеза речи и в итоге скомпилированы во фреймворк с максимально простым API и пошаговой инструкцией по реализации своих ботов.

<img src="/imgs/im2.png" width="600"/>
<img src="/imgs/im3.png" width="600"/>
<img src="/imgs/im5.png" width="600"/>

UGhost работает в веб среде, такой подход делает его **кроссплатформенным**. Фреймфорк может быть развернут и в **сетевых приложениях**, и на **локальной машине**, и, при помощи фреймворка Cordova, на **мобильных устройствах**.
